# BMI Calculator

A simple BMI Calculator web app made with HTML, CSS, and JavaScript.

## 🚀 Features
- Input height (cm) and weight (kg)
- Calculate BMI
- Show BMI category (Underweight, Normal, Overweight, Obese)
